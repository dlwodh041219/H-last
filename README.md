# Emoji Choreo (이모지 코레오) — 움직이는 나만의 이모지

카메라 속 내 얼굴을 “내가 커스텀한 이모지 얼굴”로 가리고, **몸 동작만으로 단계별 미션을 수행하는 인터랙티브 게임**입니다.  
눈/코/입/헤어/악세사리 등을 조합해 만든 이모지가 내 얼굴을 따라다니며, 안무처럼 동작 게임을 진행합니다.

## 데모 링크

## 실행 방법 (로컬 실행 권장)

### VS Code + Live Server로 실행
1. 프로젝트 **폴더 전체**를 압축 해제합니다.
2. VS Code에서 **Open Folder(폴더 열기)** 로 이 프로젝트 폴더(H-last)를 엽니다.
3. `index.html`(또는 시작 HTML 파일)을 우클릭 → **Open with Live Server**
4. 브라우저에서 카메라 권한 **허용(Allow)**

> 주의: 카메라/리소스 로드(CORS) 때문에 “파일 더블클릭(로컬 file://)”은 정상 동작이 안 될 수 있습니다.  
> 반드시 Live Server 같은 로컬 서버에서 실행하세요.

## 게임 흐름(구성)
1. **Start Page**
2. **Template 선택(게임 선택)**
3. **이모지 커스터마이징(Stage 2)**  
   - 눈/코/입/눈썹/헤어/악세사리 등을 선택해 얼굴 이모지를 구성
   - 얼굴 트래킹으로 이모지가 얼굴을 따라가도록 적용
4. **동작 게임(Stage 3)** — 선택한 템플릿에 따라 4단계 미션 수행  
   - 동물 키우기 / 요리하기 / 집 짓기 (각각 4단계 동작 미션)
5. **캡쳐 + QR(Stage 4)**  
   - 완료 순간을 촬영 → 업로드 → QR 생성(스캔하면 이미지 저장)

또한 일정 시간(약 1분 30초) 입력/동작이 없으면 **자동으로 첫 화면으로 복귀**하도록 설계했습니다.

## 템플릿(게임)별 미션 예시

### 동물 키우기
- 1단계: 안아주기  
- 2단계: 먹이주기  
- 3단계: 씻기기  
- 4단계: 놀아주기

### 요리하기
- 1단계: 재료 자르기  
- 2단계: 재료 넣기  
- 3단계: 볶기  
- 4단계: 맛보기(입 움직임 기반)

### 집 짓기
- 1단계: 도끼질  
- 2단계: 톱질  
- 3단계: 망치질  
- 4단계: 인사

## 조작 방법(UI)
- 화면의 버튼을 **마우스로 클릭**해서 진행합니다.
- 각 단계에서 일정 시간이 지나면 **SKIP(건너뛰기)** 가 활성화됩니다.
- 완료 후 셔터 버튼을 눌러 **촬영 → 프리뷰(다시찍기/QR 저장)** 로 진행합니다.

---

## 사용 기술 / 라이브러리
- **p5.js**: 전체 UI/장면 전환/렌더링
- **ml5.js**
  - Face 관련 트래킹(이모지 얼굴 적용)
  - BodyPose(MoveNet) 기반 동작 인식(동작 템플릿 미션)
- **clmtrackr**: 요리 템플릿 “맛보기” 단계에서 입 움직임 인식
- **Supabase Storage + QR 생성**
  - 캡쳐 이미지를 업로드 → public URL 생성 → QR 코드로 변환

## 실행 환경/주의사항
- 카메라 인식이 잘 되려면:
  - 모자/마스크 등 얼굴 가림 최소화
  - 조명이 너무 어둡지 않게
  - 상반신이 화면 안에 충분히 들어오게
  - 스켈레톤/키포인트가 안정적으로 잡힌 뒤 동작 수행

## 팀 / 역할
- 김정우: 이모지 커스텀(stage2) 개발, 템플릿(각 4단계) 전반 코드/디버깅, 단계별 아이콘 디자인
- 정재원: 시작 페이지 개발, 동물 키우기(1~3단계) 개발, 클리어 컷/가이드 이미지 담당
- 이다인: 프로그램 병합/디버깅, 집 짓기 템플릿 개발, 캡쳐 및 QR 페이지 개발
- 이재오: 튜토리얼 디자인/개발, 얼굴 트래킹 구현, 요리하기(1~3단계) 개발, PPT 제작
